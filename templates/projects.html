Database Server

The goal of this projects was to write a database server from scratch in the C language. Many aspects of programming were introduced including sockets, threading, parsing, and team project management.

We opted to use industry-standard protocols (JSON) and hash tables in the storage system to provide constant-time lookup.

Check out our final presentation in Chrome/Safari or Other Browsers.
